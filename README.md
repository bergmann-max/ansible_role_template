## Installation

Include via a `requirements.yml` and installed with `ansible-galaxy`:

```yaml
# requirements.yml
roles:
  - name: template
    src: git+ssh://git@github.com/bergmann-max/ansible_role_template.git
    version: main
    scm: git
```

```bash
ansible-galaxy install -r requirements.yml
```
